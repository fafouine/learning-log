 ### completed 5 levels of [bandit wargames](https://bandit.labs.overthewire.org)
- **tools used:** ssh, [overthewire.org](https://overthewire.org)
- **what I've learnt:**
  - to access the contents of a file that starts with `-`, use it's full path (e.g.: `./-` otherwise the system thinks we are inputing an option.
  - `ls -R` can be used to look inside directories recursively
  - `cat` can be used to concatenate multiple files that start the same but end differently by using wildcards (`*`)
